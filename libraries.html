A number of libraries helped with this problem set.

  • For all mathematics formatting I use the MathJax JavaScript library.
  • All code snippets use codeblock.js to make them editable and runnable, which in turn depends on JQuery and Ace.
  • Templates are done with Mustache. (see the /src directory)
  • compile.js is run with Node.js.
  • This code is hosted on BitBucket at http://primes-app-2016.bitbucket.org, and uses Git to track and sync changes.
