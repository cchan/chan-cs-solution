  1. A number of code libraries helped with this problem set.

    • For all mathematics formatting I use the MathJax JavaScript library. I used a number of various sources for LaTeX syntax lookup, omitted from here for brevity's sake.
    • All code snippets use codeblock.js to make them editable and runnable, which in turn depends on JQuery and Ace.
    • Templates are done with Mustache. (see the /src directory)
    • compile.js is run with Node.js.
    • This code is hosted on BitBucket at http://chan-cs-solution.bitbucket.org, and uses Git to track and sync changes.
